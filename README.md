# Como trabajar con git
- primero cada integrante tiene que hacer su propia rama con el comando: git checkout -b nombreintegrante.
- cada integrante solo debe trabajar en su propia rama, usa
- para guardar temporarlmente los cambios, usar el comando: git add .
- para confirmar y guardar permanenetemente los cambios en la rama: git commit -m "indicar los cambios hechos"
- para subir todos los cambios hechos a la rama, usar el comando: git push
